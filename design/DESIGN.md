# 猫叔的短剧工作台 · 重设计（暗色专业创作台）

> 产品名：**猫叔的短剧工作台**。定位：剧本到来之后的短剧生产（造型→分镜→画面→图生视频·后期）。

## 0. 双视图模型（核心交互原则）

- **时序图**：拆解分镜 / 纵览全局的最佳视图。横轴时序，纵向多轨（场景轨 + 造型轨 + 镜头轨 + 对白轨 + 就绪轨），
  一眼看清整条时序的场景与造型**连续跨度**，`▲` 标记造型/场景**变化点**（换装、受伤妆、时间跳跃、转场），用于应对过程中的变化。
- **画面工作台**：单镜头创作的最佳视图。钻进一个镜头做首/关/尾帧 + 提示词 + 检视器。
- **联动**：时序图是"地图"，双击镜头卡 → 进入画面工作台创作；工作台内切换镜头 = 沿时序前后移动。
- 造型/场景轨与镜头轨共享一份数据（延续原点"两视图共用一份对象"的思想）：在时序图上调造型/场景，画面台与生成即时跟随。

## 1. 流程重构（IA）

平台 原有 IA 割裂：分镜编辑页 / 分镜工作室 / 任务中心 三处跳来跳去，用户不知道自己在流程哪一步。
重构为**一条线性工作流 + 单一工作台**，左轨永远显示"我在第几步、下一步是什么"：

```
剧本  →  造型  →  分镜  →  画面
Script   Cast     Board    Frames
(接原点) (角色/场景 (文字分镜) (首/关/尾帧图
        道具/服装  景别机位   → 图生视频·后期)
        + 参考图)  对白)
```

- **剧本**：接原点产出的 story-bible，展示各集正文，一键推进到造型/分镜。
- **造型**：角色/演员/场景/道具/服装的设定与参考图（gpt-image-2）。一致性资产库。
- **分镜**：每集镜头列表，景别/机位/运镜/对白/出场实体确认。
- **画面**：每个镜头的首帧/关键帧/尾帧图；后期接图生视频。

顶部一条**进度脉络**始终可见（当前项目完成度）；任务状态收进右上角轻量指示器，不再单独占一块。

## 2. 视觉系统（暗色专业）

**黑金 · 好莱坞黄金时代**：暖黑为底、金作装饰。金只用于标题/细线/选中/关键操作，黑占主导（"金为点缀不铺满"）。
标题走衬线金字（Playfair Display / 思源宋体）+ 字距，呼应经典片头字幕；navy+gold 经典好莱坞配色用于场景轨。
（早期曾用琥珀橙，2026-07-02 改为更华贵的金属金 #d4af37。）

### 色板（token）
| 变量 | 值 | 用途 |
|---|---|---|
| `--bg` | `#0d0f12` | 页面底（近黑，微冷） |
| `--surface` | `#16191e` | 面板/侧轨 |
| `--surface-2` | `#1c2027` | 卡片/输入框 |
| `--surface-3` | `#22272f` | 悬浮/选中 |
| `--border` | `#282e37` | 分隔线（极克制） |
| `--text` | `#eceef1` | 主文字 |
| `--text-2` | `#9aa3af` | 次要文字 |
| `--text-3` | `#5f6875` | 占位/弱化 |
| `--accent` | `#e8a33d` | 唯一强调色（琥珀金，电影暖调） |
| `--accent-soft` | `rgba(232,163,61,.12)` | 选中底/标签 |
| `--success` | `#4ea87b` ｜ `--danger` `#d16a5a` ｜ `--info` `#5a8fd1` | 状态 |

### 其他
- 字体：`Inter / DM Sans / PingFang SC`，紧凑；仅项目大标题可选衬线点缀。
- 圆角 10px；卡片**无边框**，靠底色明度分层（surface 梯子）+ 极淡阴影。
- 间距梯子 4/8/12/16/24/32；信息密度中等偏低，留白撑起高级感。
- 强调色只用于：主操作按钮、选中态、关键数据、进度——绝不铺满。
- 状态色只做状态，不与强调色混用（延续原点"颜色各司其职"原则，但整体转暗色）。

## 3.5 分镜 = 时序图 + 卡片（参考原点排演视图）

镜头组织采用原点「剧情板·排演视图」的**时序图 + 卡片**范式（轨道 × 时序矩阵），转为暗色：
```
时序 →   建置(冷开场·对峙) │ 对抗(撕裂·退让) │ 收束(留白钩子)   ← 幕/节拍表头
场景轨   [ 总裁办公室 scene_001 ══════════ ] [ 窗外空镜 scene_002 ]  ← 同场景连续可视
镜头轨   [卡01✓][卡02✓][卡03·选中][卡04○][卡05○]                    ← 镜头卡片按时序
对白轨   陆:你终于来了 │ — │ 陆:我不会同意 │ 林:我不是她 │ —
就绪轨   ▪▪▪齐 │ ▪▪▪齐 │ ▪▪□缺尾帧 │ □□□未生成 │ □□□未生成       ← 首/关/尾帧进度
```
- 横轴 = 幕/节拍时序（建置→对抗→收束，取自 24 集短剧节拍）。
- **场景轨**用色块表达同一场景的连续镜头跨度 → 直接服务造型/一致性（同场景同造型）。
- **镜头轨**是卡片：缩略图 + 序号 + 景别 + 时长 + 状态点。
- **就绪轨**用三格 pip 表示首/关/尾帧生成进度。
- 原型见 `redesign-storyboard-timeline.html`。

## 4. 落地路径
1. ✅ 高保真原型（暗色专业）：
   - `redesign-cast.html` — **造型**（角色/演员/场景/道具/服装库 + 参考图 + 造型变化）
   - `redesign-storyboard-timeline.html` — **分镜·时序图**（场景轨+造型轨+镜头轨+对白轨+就绪轨，▲变化点）
   - `redesign-dark-pro.html` — **画面工作台**（单镜头首/关/尾帧 + 提示词 + 检视器）
   - 剧本段 = 接原点产出，仅导入+分集浏览，复杂度低，落地时直接做
2. 认可后：把设计系统 token 落进 平台（antd v5 暗色算法 `theme.darkAlgorithm` + 自定义 token + 全局皮肤）
3. 逐页重构 IA：合并"分镜编辑页/工作室"为单一工作台，任务中心降级为顶栏指示器
4. 不改后端与业务逻辑，只动前端布局与皮肤
